# locker-proj backend
ocsefhacks project


## Building/Developing
1. Clone repo
```sh
git clone https://github.com/Seirea/locker-proj
```
2. Setup virtual environment
```sh
python3 -m venv .venv
# on windows
.venv/Scripts/activate
# on *nix systems
. .venv/Scripts/activate.sh
```
3. install deps
```sh
pip3 install -r requirements.txt
```
